# Crypto Shift (暗号資産ニュースアプリ) - 仕様書

## プロジェクト概要
- **アプリ名**: Crypto Shift
- **プラットフォーム**: iOS / Android (Flutter 3.x)
- **最新バージョン**: v1.0.11+12
- **目的**: 最新の暗号資産ニュース、および主要な金融指標（暗号資産、株式指数）をユーザーに提供する総合ニュースアプリ。

## 主要機能

### 1. ニュースフィード（ホーム画面）
- **データソース**: WordPress REST API (`https://crypto-shift.com/wp-json/wp/v2/posts`)
- **UIレイアウト（Yahoo!ニュース風スタイル）**:
  - 各記事のカードは横並びレイアウトを採用。
  - 左側に「タイトル（最大3行）」「日付」「ブックマークボタン」をコンパクトに配置。
  - 右側に高画質サムネイル画像（90x90、角丸）を配置。文字数に応じてカードの高さが最適化される設計。
  - トップに「Crypto Shift」の専用ロゴとブランド名を表示。
- **既読機能**: 既読した記事はタイトル文字やサムネイルが少し暗くなり、視覚的に区別可能。

### 2. 金融市場ティッカー（カテゴリ下部表示）
- **データソース**: Binance API (暗号資産) および Alpha Vantage / 疑似データ等 (株式指数)
- **表示内容**:
  - 暗号資産: BTC, ETH, SOL
  - 国内外指数: 日経平均, JPX日経インデックス400, NYダウ, NASDAQ, S&P500, 日経300
  - 計9つの指標が画面上でスムーズに横流し（マーキー）表示される。

### 3. 記事詳細画面
- **表示内容**: WordPressのHTMLコンテンツ（見出し、本文、画像）をネイティブアプリ内で最適化して表示。
- **スタイリング調整**:
  - 暗号資産界隈のニュースに合わせた洗練されたデザイン（ダークモード/ライトモード対応）。
  - サマリー（要約）部分や見出しの文字色・フォントを本文と統一し、可読性を向上。

### 4. マイページ & 設定
- **テーマ切り替え**: ダークモード / ライトモードのワンタップ切り替え。
- **フォントサイズ変更**: 標準(1.0) → 大(1.2) → 特大(1.4) の3段階で文字サイズを一括調整。
- **法的表示（外部遷移）**:
  - 「プライバシーポリシー」および「利用規約（免責事項含む）」をアプリ内メニューからブラウザで開く機能。

### 5. その他の仕様・設定
- **ローカライゼーション**: `flutter_localizations` を導入し、Android/iOSシステム全体で正しい日本語フォント（中華フォント化の防止）を強制適用。
- **アプリアイコン**: Google Play Console用の公式ロゴ（青・緑グラデーション、Cマーク＋矢印）を全ての解像度および起動時スプラッシュ画面に完全適用済み。

## 開発・ビルド環境
- **言語/フレームワーク**: Dart / Flutter
- **主要パッケージ**: `provider`, `cached_network_image`, `flutter_html`, `shared_preferences`, `url_launcher`, `flutter_launcher_icons`, `flutter_native_splash` など。
